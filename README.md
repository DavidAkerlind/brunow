
# Brunow von Magnusson

A mobile-first web application that displays the complete rules for **"Brunow von Magnusson"**, a drinking game played with UNO cards.

---

## Features

- Clean, mobile-optimized interface  
- Complete game rules and setup instructions  
- Detailed card meanings with visual indicators  
- Toggle to view original rules image  
- Responsive design with smooth animations  
- Swedish language support  

---

## Tech Stack

- **React**
- **Fimga for designing**

---

##  Game Overview

Brunow von Magnusson is a drinking game that requires:

- A **UNO deck**  
- At least **4 players** (recommended)  
- Your **beverage of choice**

Players reveal cards simultaneously, and the player with the lowest card drinks according to card values and special rules.
